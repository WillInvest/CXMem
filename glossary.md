# Glossary

## round

A bounded unit of agent work within a session.

> [!warning] Definition may evolve in R3/R4

## session

A continuous span of work that may include many rounds and may survive `/clear` through memory.

> [!warning] Definition may evolve in R3/R4

## project-memory

Durable memory for project facts, decisions, pointers, and state that should persist across sessions.

> [!warning] Definition may evolve in R3/R4

## hour-memory

Short-horizon memory for recent work that is useful within roughly the current working block.

> [!warning] Definition may evolve in R3/R4

## day-memory

Medium-horizon memory for recent work that may guide later sessions before it is distilled or dropped.

> [!warning] Definition may evolve in R3/R4

## pointer

A reference to source material that lets an agent recover details without copying the whole source into memory.

> [!warning] Definition may evolve in R3/R4

## distillation

Compression of noisy work history into concise memory that preserves the useful outcome and enough evidence to audit it.

> [!warning] Definition may evolve in R3/R4

## auditor

A review role that checks memory changes for drift, quality, evidence, and usefulness.

> [!warning] Definition may evolve in R3/R4

## intention

The agent's stated purpose for an action, tool call, memory entry, or next step.

> [!warning] Definition may evolve in R3/R4
